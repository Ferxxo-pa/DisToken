Build the following three features for this DisToken NFT display app. This is a dark-themed React/Vite/TypeScript app using Tailwind CSS v4, wouter for routing, and lucide-react for icons.

## Feature 1: Floor Price in nft.ts

Add a `fetchFloorPrice(contractAddress: string): Promise<{ eth?: number } | null>` function to `src/lib/nft.ts`.

Implementation:
- Use Reservoir API: `https://api.reservoir.tools/collections/v6?id={contractAddress}&limit=1`
- Auth header: `x-api-key: demo`
- Parse `data.collections[0]?.floorAsk?.price?.amount?.decimal` for ETH price
- Cache in localStorage with key `floorprice_${contractAddress}` and 5-minute TTL (store as JSON with `{price, ts}`)
- Return null if fetch fails or no data

## Feature 2: Floor Price Badge on NFTSlideshow

In `src/components/NFTSlideshow.tsx`:
- Import and call `fetchFloorPrice` with the contractAddress of the currently displayed NFT
- Store floor price in state as a number or null
- Show a pill badge in the bottom-right of the slideshow container when floor price is available
- Badge style: `position: absolute, bottom: 12px, right: 12px` — `bg-black/60 backdrop-blur-sm rounded-full px-3 py-1 text-xs text-white font-mono`
- Content: `E {eth.toFixed(3)}` (use the unicode Ξ symbol: \u039E)
- Only fetch once per unique contractAddress (dedupe with useRef or useEffect dep)
